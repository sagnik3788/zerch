# Zerch-pvt

A vector search engine for logs.

## Architecture(will update)

```
Zerch-pvt/
├── crates/
│   ├── zerch-pvt-core      
│   ├── zerch-pvt-embed     
│   ├── zerch-pvt-storage   
│   └── zerch-pvt-engine    
```

## Getting Started

Build the project:

```bash
cargo build
```

Run the engine:

```bash
cargo run -p zerch-engine
```
